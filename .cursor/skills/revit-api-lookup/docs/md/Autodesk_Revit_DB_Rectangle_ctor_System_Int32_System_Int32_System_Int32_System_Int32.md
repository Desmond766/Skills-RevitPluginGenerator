# Autodesk.Revit.DB.Rectangle.#ctor Method

Creates an Rectangle instance with the specified four integers
 that represent the left, top, right and bottom of a rectangle.

## Syntax (C#)
```csharp
public Rectangle(
	int left,
	int top,
	int right,
	int bottom
)
```

## Parameters
- **left** (`System.Int32`) - The x-coordinate of the top-left corner.
- **top** (`System.Int32`) - The y-coordinate of the top-left corner.
- **right** (`System.Int32`) - The x-coordinate of the bottom-right corner.
- **bottom** (`System.Int32`) - The y-coordinate of the bottom-right corner.
