# Autodesk.Revit.UI.FileDialog.GetSelectedModelPath Method

Returns the selected file path chosen by the user.

## Syntax (C#)
```csharp
public ModelPath GetSelectedModelPath()
```

## Returns
The selected file path, or Nothing nullptr a null reference ( Nothing in Visual Basic) if the dialog has not been shown or selection was cancelled.
