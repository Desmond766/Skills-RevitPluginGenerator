# Autodesk.Revit.DB.Analysis.MassSurfaceData.SkylightWidth Property

The length dimension to be used for the sides of each individual square skylight
 produced in the grid of auto-generated skylights.

## Syntax (C#)
```csharp
public double SkylightWidth { get; set; }
```

## Exceptions
- **Autodesk.Revit.Exceptions.ArgumentOutOfRangeException** - When setting this property: The skylight width is less than eight inches.
