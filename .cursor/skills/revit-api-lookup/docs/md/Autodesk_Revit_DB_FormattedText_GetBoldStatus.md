# Autodesk.Revit.DB.FormattedText.GetBoldStatus Method

Returns whether All , None or a Mixed of characters in the entire text are bold.

## Syntax (C#)
```csharp
public FormatStatus GetBoldStatus()
```

## Returns
The format status of bold on characters FormatStatus .

## Remarks
This function only returns All or None if the entire text contains one character.
