# Autodesk.Revit.DB.FormatOptions.CanUsePlusPrefix Method

Checks whether a plus prefix can be displayed for the display unit in this FormatOptions.

## Syntax (C#)
```csharp
public bool CanUsePlusPrefix()
```

## Returns
True if a plus prefix can be displayed, false otherwise.

## Exceptions
- **Autodesk.Revit.Exceptions.InvalidOperationException** - UseDefault is true in this FormatOptions.
