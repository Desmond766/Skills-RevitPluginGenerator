# Autodesk.Revit.DB.Structure.BendingDetailCustomFieldProperties.SegmentLengthDimensionsForHooksEnabled Property

Identifies if segment length dimensions for hooks will be shown or not.
 If this property is true, the SegmentLengthDimensionsEnabled should also be true to see segment length dimensions for hooks.

## Syntax (C#)
```csharp
public bool SegmentLengthDimensionsForHooksEnabled { get; set; }
```
