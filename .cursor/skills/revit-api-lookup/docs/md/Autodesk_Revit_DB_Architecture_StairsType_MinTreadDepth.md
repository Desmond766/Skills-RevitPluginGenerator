# Autodesk.Revit.DB.Architecture.StairsType.MinTreadDepth Property

The minimum tread width along the center path for all common runs (winder, arc, and straight).

## Syntax (C#)
```csharp
public double MinTreadDepth { get; set; }
```

## Exceptions
- **Autodesk.Revit.Exceptions.ArgumentOutOfRangeException** - When setting this property: The given value for minTreadDepth must be greater than 0 and no more than 30000 feet.
