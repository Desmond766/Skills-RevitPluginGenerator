## Syntax (C#)


## Remarks
User can select the candidate via PaperSources and then 
set the Page source of the Print Setting.

## Exceptions

- **Autodesk.Revit.Exceptions.ArgumentException** - Thrown when the input argument is invalid for current printer.
