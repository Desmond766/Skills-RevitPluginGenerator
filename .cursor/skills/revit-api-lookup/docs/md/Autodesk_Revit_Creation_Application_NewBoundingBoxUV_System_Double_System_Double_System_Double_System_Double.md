# Autodesk.Revit.Creation.Application.NewBoundingBoxUV Method

## Syntax (C#)
```csharp
public BoundingBoxUV NewBoundingBoxUV(
	double min_u,
	double min_v,
	double max_u,
	double max_v
)
```

## Parameters
- **min_u** (`System.Double`) - The first coordinate of min.
- **min_v** (`System.Double`) - The second coordinate of min.
- **max_u** (`System.Double`) - The first coordinate of max.
- **max_v** (`System.Double`) - The second coordinate of max.
