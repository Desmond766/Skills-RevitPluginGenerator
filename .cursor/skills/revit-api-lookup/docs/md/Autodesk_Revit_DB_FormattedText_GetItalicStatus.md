# Autodesk.Revit.DB.FormattedText.GetItalicStatus Method

Returns whether All , None or a Mixed of characters in the entire text are italic.

## Syntax (C#)
```csharp
public FormatStatus GetItalicStatus()
```

## Returns
The format status of italic on characters FormatStatus .

## Remarks
This function only returns All or None if the entire text contains one character.
