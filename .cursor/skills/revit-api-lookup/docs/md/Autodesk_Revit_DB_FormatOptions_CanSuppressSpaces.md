# Autodesk.Revit.DB.FormatOptions.CanSuppressSpaces Method

Checks whether spaces can be suppressed for the display unit in this FormatOptions.

## Syntax (C#)
```csharp
public bool CanSuppressSpaces()
```

## Returns
True if spaces can be suppressed, false otherwise.

## Exceptions
- **Autodesk.Revit.Exceptions.InvalidOperationException** - UseDefault is true in this FormatOptions.
