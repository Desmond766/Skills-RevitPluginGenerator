# Autodesk.Revit.DB.HermiteSurface.IsValid Method

Checks whether this HermiteSurface object is valid.

## Syntax (C#)
```csharp
public bool IsValid()
```

## Returns
True if this is a valid Hermite Surface, false otherwise.

## Remarks
