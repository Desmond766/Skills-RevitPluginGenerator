# Autodesk.Revit.UI.PulldownButton

The PulldownButton object represents a button with a drop-down list on RibbonPanel.

## Syntax (C#)
```csharp
public class PulldownButton : RibbonButton
```

## Remarks
Use of this class is not supported in Revit Macros.
