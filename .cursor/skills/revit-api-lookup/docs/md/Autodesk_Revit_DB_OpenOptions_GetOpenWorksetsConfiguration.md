# Autodesk.Revit.DB.OpenOptions.GetOpenWorksetsConfiguration Method

Gets the object used to configure the worksets to open when the model is opened.

## Syntax (C#)
```csharp
public WorksetConfiguration GetOpenWorksetsConfiguration()
```

## Returns
The options. If Nothing nullptr a null reference ( Nothing in Visual Basic) , all user-created worksets will be opened.
