# Autodesk.Revit.DB.ImageTypeStatus.FailedToLoad

The linked image could not be loaded from file. 
 Some possible reasons are:
 The file could not be found The file could be corrupt or not contain image data In case of PDF files, the file might be encrypted or not have the requested page
