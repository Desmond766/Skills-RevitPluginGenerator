# Autodesk.Revit.DB.Structure.Rebar.GetFreeFormAccessor Method

```csharp
public RebarFreeFormAccessor GetFreeFormAccessor()
```

## Returns
The interface providing access to free-form properties and methods for this Rebar element.

## Exceptions
