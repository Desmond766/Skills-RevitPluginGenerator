# Autodesk.Revit.Exceptions.InsufficientResourcesException

The exception that is thrown when the OS runs out of resources, 
e.g. memory, disk space, or USER or GDI objects.

## Syntax (C#)
```csharp
[SerializableAttribute]
public class InsufficientResourcesException : InvalidOperationException
```
