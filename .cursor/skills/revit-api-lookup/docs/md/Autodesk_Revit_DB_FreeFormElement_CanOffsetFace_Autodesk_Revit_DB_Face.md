# Autodesk.Revit.DB.FreeFormElement.CanOffsetFace Method

Determines if the input face of the FreeFormElement can be offset.

## Syntax (C#)
```csharp
public bool CanOffsetFace(
	Face face
)
```

## Parameters
- **face** (`Autodesk.Revit.DB.Face`) - The face to be checked.

## Returns
True if the face can be offset, false otherwise.

## Exceptions
- **Autodesk.Revit.Exceptions.ArgumentNullException** - A non-optional argument was null
