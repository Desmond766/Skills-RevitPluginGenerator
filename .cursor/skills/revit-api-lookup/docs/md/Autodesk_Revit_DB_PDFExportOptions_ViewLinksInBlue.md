# Autodesk.Revit.DB.PDFExportOptions.ViewLinksInBlue Property

Whether set view links blue.

## Syntax (C#)
```csharp
public bool ViewLinksInBlue { get; set; }
```

## Remarks
If true, links will be visible as blue.
 If false, they retain their default text color.
