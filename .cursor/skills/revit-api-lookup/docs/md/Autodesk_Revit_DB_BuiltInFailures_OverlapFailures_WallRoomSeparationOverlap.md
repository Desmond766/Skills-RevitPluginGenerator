# Autodesk.Revit.DB.BuiltInFailures.OverlapFailures.WallRoomSeparationOverlap Property

A wall and a room separation line overlap. One of them may be ignored when Revit finds room boundaries.
 Shorten or delete the room separation line to remove the overlap.
