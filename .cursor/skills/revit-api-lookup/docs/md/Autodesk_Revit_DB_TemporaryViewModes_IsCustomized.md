# Autodesk.Revit.DB.TemporaryViewModes.IsCustomized Method

Identifies if a custom temporary view mode is currently active.
 A custom mode is active if there is a non-empty string set for CustomTitle .

## Syntax (C#)
```csharp
public bool IsCustomized()
```

## Returns
Returns true a custom temporary view mode is currently active, false otherwise.
 A custom mode is active if there is a non-empty string set for CustomTitle .
