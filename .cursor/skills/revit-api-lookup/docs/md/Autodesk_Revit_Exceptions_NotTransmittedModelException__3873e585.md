# Autodesk.Revit.Exceptions.NotTransmittedModelException

The exception thrown when OpenOptions were provided to deal with
a transmitted model, but the model is not transmitted.

## Syntax (C#)
```csharp
[SerializableAttribute]
public class NotTransmittedModelException : InvalidOperationException
```
