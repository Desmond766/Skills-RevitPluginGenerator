# Autodesk.Revit.DB.Structure.RebarContainerItem.IsRebarInSection Method

Identifies if this rebar set is shown as a cross-section in the given view.

```csharp
public bool IsRebarInSection(
	View dBView
)
```

## Parameters
- **dBView** (`Autodesk.Revit.DB.View`) - The view.
