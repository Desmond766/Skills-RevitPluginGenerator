# Autodesk.Revit.DB.Fabrication.FabricationPartRouteEnd

Class to hold fabrication part routing start or end information.

## Syntax (C#)
```csharp
public class FabricationPartRouteEnd : IDisposable
```

## Remarks
To create a new instance, use CreateFromConnector(Connector) to specify a connector,
 or CreateFromCenterline(Element, XYZ) to specify the centerline of a straight element,
 for the fabrication routing end point.
