# Autodesk.Revit.DB.ParameterMap.Contains Method

Tests for the existence of a key within the map.

## Syntax (C#)
```csharp
public virtual bool Contains(
	string key
)
```

## Parameters
- **key** (`System.String`) - The key to be searched for.

## Returns
The Contains method returns True if the key is within the map, otherwise False.
