# Autodesk.Revit.DB.CylindricalSurface

A cylindrical surface.

## Syntax (C#)
```csharp
public class CylindricalSurface : Surface
```

## Remarks
The parametric equation of the cylinder is S(u, v) = center + radius*cos(u)*xVec + radius*sin(u)*yVec + v*zVec.
