# Autodesk.Revit.UI.TabbedDialogExtension.GetContextualHelp Method

Gets the contextual help.

## Syntax (C#)
```csharp
public ContextualHelp GetContextualHelp()
```

## Returns
The contextual help assigned to the help button of the Revit options dialog, or Nothing nullptr a null reference ( Nothing in Visual Basic) if there is no binding assigned.
