# Autodesk.Revit.DB.Infrastructure.Alignment.StartStation Property

Specifies the station at which the alignment's geometric definition starts, in Revit internal model units (standard Imperial feet).

## Syntax (C#)
```csharp
public double StartStation { get; }
```

## Remarks
This station may be outside the displayed range ( DisplayedStartStation , DisplayedEndStation ).
