# Autodesk.Revit.DB.BaseImportOptions.ColorMode Property

Color mode for the import.
 Three modes are supported. Black and White, Preserve Colors, and Invert Colors.

## Syntax (C#)
```csharp
public ImportColorMode ColorMode { get; set; }
```
