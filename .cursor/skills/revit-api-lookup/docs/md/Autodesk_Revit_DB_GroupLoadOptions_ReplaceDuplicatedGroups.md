# Autodesk.Revit.DB.GroupLoadOptions.ReplaceDuplicatedGroups Property

If there are groups with the same names in source and destination documents set this property to True true true ( True in Visual Basic) to replace existing groups,
 otherwise the operation will be canceled. The default value is False false false ( False in Visual Basic) .

public bool ReplaceDuplicatedGroups { get; set; }
