# Autodesk.Revit.DB.BuiltInFailures.OverlapFailures.WallSpaceSeparationOverlap Property

A wall and a space separation line overlap. One of them may be ignored when Revit finds space boundaries.
 Shorten or delete the space separation line to remove the overlap.
