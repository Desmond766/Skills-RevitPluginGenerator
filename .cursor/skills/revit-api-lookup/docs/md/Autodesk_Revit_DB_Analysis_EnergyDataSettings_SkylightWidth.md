# Autodesk.Revit.DB.Analysis.EnergyDataSettings.SkylightWidth Property

Used for the conceptual energy model. The approximate width used for the skylights in
 massing instances when the Energy Analytical model is being created.

## Syntax (C#)
```csharp
public double SkylightWidth { get; set; }
```

## Exceptions
- **Autodesk.Revit.Exceptions.ArgumentOutOfRangeException** - When setting this property: The skylight width is less than eight inches.
