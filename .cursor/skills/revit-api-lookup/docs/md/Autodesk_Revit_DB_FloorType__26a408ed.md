# Autodesk.Revit.DB.FloorType

An object that specifies the type of a floor in Autodesk Revit.

## Syntax (C#)
```csharp
public class FloorType : HostObjAttributes
```

## Remarks
The structural layers of the floor can be accessed via this object.
