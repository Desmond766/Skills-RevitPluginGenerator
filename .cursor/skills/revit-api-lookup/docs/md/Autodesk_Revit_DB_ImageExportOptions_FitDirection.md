# Autodesk.Revit.DB.ImageExportOptions.FitDirection Property

The fit direction. Used only if ZoomType is FitToPage.

## Syntax (C#)
```csharp
public FitDirectionType FitDirection { get; set; }
```

## Remarks
The default is Horizontal.

## Exceptions
