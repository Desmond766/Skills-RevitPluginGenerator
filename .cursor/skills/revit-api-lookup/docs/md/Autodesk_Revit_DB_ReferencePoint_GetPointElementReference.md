# Autodesk.Revit.DB.ReferencePoint.GetPointElementReference Method

Retrieve a copy of the rule that computes the
location of the ReferencePoint relative to other elements in
the document.

## Syntax (C#)
```csharp
public PointElementReference GetPointElementReference()
```

## Returns
A PointElementReference object, or Nothing nullptr a null reference ( Nothing in Visual Basic) if the
ReferencePoint does not have a reference.
