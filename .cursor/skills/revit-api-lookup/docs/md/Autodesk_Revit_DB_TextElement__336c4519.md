# Autodesk.Revit.DB.TextElement

Base class representing text elements in Revit.

## Syntax (C#)
```csharp
public class TextElement : Element
```

## Remarks
TextElement is a base class for other annotation classes,
 like TextNote, which have additional, more specific functionality.
