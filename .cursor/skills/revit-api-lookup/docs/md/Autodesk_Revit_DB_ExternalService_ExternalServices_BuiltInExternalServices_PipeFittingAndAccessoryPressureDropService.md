# Autodesk.Revit.DB.ExternalService.ExternalServices.BuiltInExternalServices.PipeFittingAndAccessoryPressureDropService Property

The external service which permits registration of an alternate implementation for pipe fitting and accessory pressure drop calculation.
 To use this service, programmers implement a server class that derives from IPipeFittingAndAccessoryPressureDropServer .
