# Autodesk.Revit.DB.FailureSeverity.Error

**中文**: 错误、报错

Failure that cannot be ignored.
 If FailureMeassage of this severity is posted, the current transaction cannot be committed unless
 failure is resolved via appropriate FailureResolution.
