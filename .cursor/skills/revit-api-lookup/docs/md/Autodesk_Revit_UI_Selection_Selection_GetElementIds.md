# Autodesk.Revit.UI.Selection.Selection.GetElementIds Method

Returns the ids of the elements that are currently selected within the project.
 The selection may not be complete. See GetReferences for more options.

## Syntax (C#)
```csharp
public ICollection<ElementId> GetElementIds()
```

## Returns
The collection containing the ids of the selected elements.
