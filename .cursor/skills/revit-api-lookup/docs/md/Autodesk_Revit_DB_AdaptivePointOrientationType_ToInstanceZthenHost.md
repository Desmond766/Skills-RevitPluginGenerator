# Autodesk.Revit.DB.AdaptivePointOrientationType.ToInstanceZthenHost

The Z axis of the adaptive point will be oriented to the Z axis of the adaptive family instance
 the point is part of. The XY axis are determined by the XY of the geometry the point is hosted on. This option was previously named: FamilyVertical
