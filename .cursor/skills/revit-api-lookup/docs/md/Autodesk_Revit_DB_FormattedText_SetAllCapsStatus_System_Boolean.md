# Autodesk.Revit.DB.FormattedText.SetAllCapsStatus Method

Sets the characters in the entire text to be in all caps or not.

## Syntax (C#)
```csharp
public void SetAllCapsStatus(
	bool isAllCaps
)
```

## Parameters
- **isAllCaps** (`System.Boolean`) - The desired all caps status of characters in the entire text.
 True to set all capped, false to set not all capped.
