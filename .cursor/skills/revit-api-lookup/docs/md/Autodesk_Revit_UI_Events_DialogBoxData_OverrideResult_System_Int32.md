# Autodesk.Revit.UI.Events.DialogBoxData.OverrideResult Method

Call this method to cause the Autodesk Revit dialog to be dismissed with the specified return value.

## Syntax (C#)
```csharp
public bool OverrideResult(
	int result
)
```

## Parameters
- **result** (`System.Int32`) - The result code you wish the Revit dialog to return.

## Returns
Returns true if the result code was accepted.
