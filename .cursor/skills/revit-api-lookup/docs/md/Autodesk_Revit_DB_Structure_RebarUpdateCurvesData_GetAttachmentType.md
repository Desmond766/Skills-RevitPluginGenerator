# Autodesk.Revit.DB.Structure.RebarUpdateCurvesData.GetAttachmentType Method

Returns attachment type for stirrups to be used in cover calculation.

## Syntax (C#)
```csharp
public StirrupTieAttachmentType GetAttachmentType()
```

## Returns
InteriorFace : rebar to attach to interior face of cover reference. ExteriorFace : rebar to attach to exterior face of cover reference.
