# Autodesk.Revit.DB.STLExportOptions.IsValidForNormalTolerance Method

```csharp
public static bool IsValidForNormalTolerance(
	double value
)
```

## Parameters
- **value** (`System.Double`) - The value to be checked.

## Returns
