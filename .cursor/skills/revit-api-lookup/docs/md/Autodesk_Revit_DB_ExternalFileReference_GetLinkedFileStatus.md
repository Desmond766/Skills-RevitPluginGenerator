# Autodesk.Revit.DB.ExternalFileReference.GetLinkedFileStatus Method

Returns a LinkedFileStatus.Enum corresponding to the
 load status of the referenced file.

## Syntax (C#)
```csharp
public LinkedFileStatus GetLinkedFileStatus()
```

## Remarks
Options are Invalid,
 Loaded, Unloaded, NotFound, or Imported.
