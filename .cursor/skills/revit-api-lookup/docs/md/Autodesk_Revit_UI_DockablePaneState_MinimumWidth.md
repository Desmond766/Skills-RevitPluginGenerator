# Autodesk.Revit.UI.DockablePaneState.MinimumWidth Property

When %dockPosition% is anything other than floating, the minimum width to use for the pane. Default is 200 pixels.

## Syntax (C#)
```csharp
public int MinimumWidth { get; set; }
```

## Exceptions
- **Autodesk.Revit.Exceptions.ArgumentOutOfRangeException** - When setting this property: The given value for minimumWidth is not positive.
