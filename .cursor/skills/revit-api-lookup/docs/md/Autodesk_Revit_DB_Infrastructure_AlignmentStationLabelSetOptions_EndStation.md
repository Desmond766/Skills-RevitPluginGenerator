# Autodesk.Revit.DB.Infrastructure.AlignmentStationLabelSetOptions.EndStation Property

The ending station for creating labels in the set, in Revit internal model units (standard Imperial feet).
 The default value is null.
 When null, the station value corresponding to the alignment's DisplayedEndStation is used.

```csharp
public Nullable<double> EndStation { get; set; }
```
