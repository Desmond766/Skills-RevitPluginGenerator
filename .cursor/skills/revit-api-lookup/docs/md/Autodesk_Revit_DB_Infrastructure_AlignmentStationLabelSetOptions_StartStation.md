# Autodesk.Revit.DB.Infrastructure.AlignmentStationLabelSetOptions.StartStation Property

The starting station for creating labels in the set, in Revit internal model units (standard Imperial feet).
 The default value is null.
 When null, the station value corresponding to the alignment's DisplayedStartStation is used.

## Syntax (C#)
```csharp
public Nullable<double> StartStation { get; set; }
```
