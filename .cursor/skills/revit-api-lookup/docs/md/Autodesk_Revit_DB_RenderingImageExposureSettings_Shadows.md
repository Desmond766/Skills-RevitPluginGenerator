# Autodesk.Revit.DB.RenderingImageExposureSettings.Shadows Property

The shadows value.

## Syntax (C#)
```csharp
public double Shadows { get; set; }
```

## Remarks
Increase this value to get darker shadows.

## Exceptions
- **Autodesk.Revit.Exceptions.ArgumentOutOfRangeException** - When setting this property: The value of shadows is not valid. The valid range is 0.1 to 4.
