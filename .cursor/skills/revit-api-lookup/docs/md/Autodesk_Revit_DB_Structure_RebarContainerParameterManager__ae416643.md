# Autodesk.Revit.DB.Structure.RebarContainerParameterManager

Provides implementation of RebarContainer parameters overrides.

## Syntax (C#)
```csharp
public class RebarContainerParameterManager : IDisposable
```

## Remarks
When a new override is created, by default, the parameter will show the overridden value as read-only.
 You can control whether or not the parameter is modifiable using SetOverriddenParameterReadonly(ElementId) and SetOverriddenParameterModifiable(ElementId) .a
