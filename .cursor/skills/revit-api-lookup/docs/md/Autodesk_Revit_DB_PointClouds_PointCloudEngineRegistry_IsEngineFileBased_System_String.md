# Autodesk.Revit.DB.PointClouds.PointCloudEngineRegistry.IsEngineFileBased Method

Identifies if a given engine is file-based.

## Syntax (C#)
```csharp
public static bool IsEngineFileBased(
	string identifier
)
```

## Parameters
- **identifier** (`System.String`) - The engine identifier.

## Returns
True if the engine is file-based, false otherwise.
