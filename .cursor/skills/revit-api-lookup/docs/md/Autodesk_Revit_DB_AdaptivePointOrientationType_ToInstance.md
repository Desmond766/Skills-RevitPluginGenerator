# Autodesk.Revit.DB.AdaptivePointOrientationType.ToInstance

The adaptive point will be oriented to the coordinate system of the adaptive
 family instance the point is part of. This is the default orientation of adaptive points. This option was previously named: FamilyOrthogonal
