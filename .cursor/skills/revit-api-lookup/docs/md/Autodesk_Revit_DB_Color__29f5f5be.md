# Autodesk.Revit.DB.Color

**中文**: 着色、上色、染色、填色

Represents a color in Autodesk Revit.

## Syntax (C#)
```csharp
public class Color : APIObject
```

## Remarks
Colors obtained from some API methods may represent an uninitialized/invalid color. You can identify these
using the IsValid property.
