# Autodesk.Revit.DB.StructuralAsset.ConcreteBendingReinforcement Property

The bending reinforcement of the asset.

## Syntax (C#)
```csharp
public double ConcreteBendingReinforcement { get; set; }
```

## Remarks
Applies to concrete-based structural assets.

## Exceptions
- **Autodesk.Revit.Exceptions.InapplicableDataException** - When setting this property: the material type must be concrete to set this property.
