# Autodesk.Revit.DB.WallType.ThermalProperties Property

The calculated and settable thermal properties of the WallType

## Syntax (C#)
```csharp
public ThermalProperties ThermalProperties { get; }
```

## Remarks
Returns Nothing nullptr a null reference ( Nothing in Visual Basic) if the wall has no thermal properties.
 Curtain walls and stacked walls do not store thermal properties.
