# Autodesk.Revit.DB.Architecture.TopographySurface.AsSiteSubRegion Method

Obtains the subregion object represented by this element.

## Syntax (C#)
```csharp
public SiteSubRegion AsSiteSubRegion()
```

## Returns
The SiteSubRegion element. If this does not represent a SiteSubRegion, this will be Nothing nullptr a null reference ( Nothing in Visual Basic) .

This applies to TopographySurface and SiteSubRegion elements.
